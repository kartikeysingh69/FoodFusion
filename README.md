# 🍔 FoodFusion

A modern Restaurant Website built using Flask.

## Features

- Responsive Design
- Hero Section
- About
- Menu
- Gallery
- Chef
- Reviews
- Contact Form
- Newsletter
- Reservation
- FAQ
- Google Map

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Python
- Flask

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```
